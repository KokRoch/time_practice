# Прогнозирование и обнаружение аномалий во временных рядах

Данный проект представляет собой учебный прототип системы для прогнозирования и обнаружения аномалий во временных рядах. В качестве данных используется набор **Numenta Anomaly Benchmark (NAB)** – общедоступный бенчмарк с размеченными временными рядами из различных предметных областей.

## Ссылка на датасет

Исходные данные доступны по ссылке:  
[https://www.kaggle.com/datasets/boltzmannbrain/nab](https://www.kaggle.com/datasets/boltzmannbrain/nab)

## Структура репозитория

- **`временные_ряды.ipynb`**  
  Jupyter Notebook с полным кодом проекта:
  - загрузка и предварительная обработка данных;
  - визуальный анализ временного ряда;
  - построение моделей прогнозирования (линейная регрессия и LSTM);
  - оценка качества моделей (MAE, RMSE);
  - обнаружение аномалий методом межквартильного размаха (IQR);
  - визуализация результатов.

- **`requirements.txt`**  
  Список необходимых Python-библиотек с версиями (см. ниже).

- **`report.pdf`**  
  Полный отчёт о ходе и результатах учебной практики (в формате PDF).

## Требования к окружению

Для воспроизведения вычислений необходима установка пакетов из `requirements.txt`
